# MISSION SPEC: Fábrica Web Premium V2 (Agentic Orchestration)

## 🎯 Objetivo General
Transformar este repositorio en la "Línea de Ensamblaje" oficial capaz de producir sitios web de élite para PyMEs de forma asíncrona y automatizada por enjambres de agentes.

## 📐 Arquitectura de Control (Protocolo SDD)
1. **Source of Truth**: Toda modificación debe basarse en archivos `.md` de especificación en la carpeta `specs/`.
2. **Componentes Atómicos**: El sitio debe fragmentarse en secciones inyectables para que múltiples agentes puedan trabajar en paralelo (ej. un agente en el Hero, otro en el catálogo).
3. **Validación Automática**: Antes de cada actualización de `index.html`, el código debe ser auditado contra las métricas de rendimiento 2026 (INP < 200ms).

## 🚀 Capacidades Especiales (Hacks de Especialista)
- **Cerebro Visual**: Inyección obligatoria de shaders WebGL en el Hero.
- **Cerebro Conversión**: Todo CTA debe pasar por la heurística de "Nivel Conductual" de Don Norman.
- **Agente de Navegador**: Se usará el `browser_subagent` para capturar capturas de pantalla de cada sección tras su edición.

## 🛠️ Plan de Implementación (Fases)
1. **Fase 1 (Limpieza)**: Estructurar carpeta `specs/` y `components/`.
2. **Fase 2 (Agentificación)**: Crear el archivo `AGENTS.md` con las reglas de orquestación.
3. **Fase 3 (Enriquecimiento)**: Inyectar las heurísticas obtenidas de NotebookLM.
